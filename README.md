# xpd
## 概要
ジムリーダーの城のパーティ構築ツールを拡張するGreasemonkey拡張スクリプトです。

## 動作環境

- [Greasemonkey 4](https://addons.mozilla.org/ja/firefox/addon/greasemonkey/)
